#### Como Usar:
1. Seleccionar una imagen
2. Pulsar "Obtener ingredientes" y esperar a que la interfaz muestre los ingredientes de la imagen
3. Repetir paso 1 y 2 hasta tener una lista de ingredientes para cada comida de tu semana
4. Pulsar "Analizar Alimentación"
5. Ver la consola del frontend para obtener tu resultado (pendiente renderizar el análisis en la interfaz)
